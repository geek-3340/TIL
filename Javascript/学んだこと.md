## ESmodules

### 基本構文<br>
以下のようにエントリーポイントのファイルで処理をエクスポートして<br>
呼び出し元のファイルでインポート・関数呼び出しを行う<br>
- エントリーポイント<br>
（例）export const hum = () => { moduleにするロジック };<br>
<br>
- 出力側<br>
（例）import { hum } from ' ./modules/modules.js ';<br>
　　　hum();
